# stf_disable_feature

Disables the specified feature flag(s) in a trace

```
usage: stf_disable_feature [options] <infile> <outfile> <features>
    -h, --help        show this message
    -V, --version     show the STF version the tool is built with
    -f                Overwrite existing file
    <infile>          Original STF file
    <outfile>         Output STF file
    <features>        Feature(s) to disable
```
