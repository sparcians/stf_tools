# stf_count_multi_process

Counts the number of processes present in a trace

```
usage: stf_count_multi_process [options] <trace>
    -h, --help        show this message
    -V, --version     show the STF version the tool is built with
    -a                Print counts for instructions before the first user process
    -v                Print which traces contain each process
    -j                Output in JSON format
    <trace>           trace in STF format
```
