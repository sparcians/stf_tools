# stf_bt9

Generates a BT9 branch trace (used in the [CBP-5 branch predictor competition](https://jilp.org/cbp2016/))

```
usage: stf_bt9 [options] <trace> <output>
    -h, --help        show this message
    -V, --version     show the STF version the tool is built with
    -u                skip non user-mode instructions
    <trace>           trace in STF format
    <output>          BT9 output filename
```
