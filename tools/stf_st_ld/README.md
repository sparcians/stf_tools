# stf_st_ld

Counts store-load forwarding opportunities in a trace

```
usage: stf_st_ld [options] <trace>
    -h, --help           show this message
    -V, --version        show the STF version the tool is built with
    -w <window_size>     window size (in # instructions)
    -v                   verbose output
    -u                   skip non user-mode instructions
    -a <alignment>       align addresses to the specified number of bytes
    <trace>              trace in STF format
```
