# stf_transaction_dump

Dumps a transaction trace in a human-readable format

```
usage: stf_transaction_dump [options] <trace>
    -h, --help        show this message
    -V, --version     show the STF version the tool is built with
    -s <N>            start dumping at N-th transaction
    -e <M>            end dumping at M-th transaction
    <trace>           trace in STF format
```
