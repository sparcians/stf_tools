# stf_isa_edit

Creates or modifies the ISA string record in a trace

```
usage: stf_isa_edit [options] <infile> <outfile>
    -h, --help        show this message
    -V, --version     show the STF version the tool is built with
    -f                Overwrite existing file
    -i <isa>          New ISA extended info string
    -d                Delete ISA extended info
    <infile>          STF to edit
    <outfile>         Output STF file
```
